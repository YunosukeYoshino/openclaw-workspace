# baseball-rehabilitation-agent

## 概要
野球リハビリ管理エージェント。選手のリハビリテーション・回復管理。

## カテゴリ
野球選手健康・メンタル

## トリガーワード
リハビリ, 回復管理, リハビリテーション

## 主な機能

### データ管理
- baseball-rehabilitation-agent 関連データのSQLiteデータベース管理
- CRUD操作の実装
- 検索・フィルタリング機能

### チャットボット機能
- Discord連携によるインタラクティブ応答
- 自然言語によるクエリ処理
- コマンドパターンマッチング

## 使用方法

### インストール
```bash
cd agents/baseball-rehabilitation-agent
pip install -r requirements.txt
```

### 実行
```bash
python agent.py
```
