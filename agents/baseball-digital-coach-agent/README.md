# baseball-digital-coach-agent

## 概要
野球デジタルコーチエージェント。デジタルツールを活用したコーチング・指導の管理。

## カテゴリ
野球デジタル・スマートスタジアム

## トリガーワード
デジタルコーチ, 指導デジタル, コーチングAI

## 主な機能

### データ管理
- baseball-digital-coach-agent 関連データのSQLiteデータベース管理
- CRUD操作の実装
- 検索・フィルタリング機能

### チャットボット機能
- Discord連携によるインタラクティブ応答
- 自然言語によるクエリ処理
- コマンドパターンマッチング

## 使用方法

### インストール
```bash
cd agents/baseball-digital-coach-agent
pip install -r requirements.txt
```

### 実行
```bash
python agent.py
```
