# baseball-team-hr-agent

## 概要
野球チーム人事エージェント。チーム人事の管理・採用。

## カテゴリ
野球チーム戦略・オペレーション

## トリガーワード
チーム人事, 人事管理, 採用

## 主な機能

### データ管理
- baseball-team-hr-agent 関連データのSQLiteデータベース管理
- CRUD操作の実装
- 検索・フィルタリング機能

### チャットボット機能
- Discord連携によるインタラクティブ応答
- 自然言語によるクエリ処理
- コマンドパターンマッチング

## 使用方法

### インストール
```bash
cd agents/baseball-team-hr-agent
pip install -r requirements.txt
```

### 実行
```bash
python agent.py
```
