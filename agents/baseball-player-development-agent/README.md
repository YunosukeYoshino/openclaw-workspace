# baseball-player-development-agent

## 概要
野球選手育成プログラムエージェント。選手育成プログラムの管理・実施。

## カテゴリ
野球選手キャリア管理

## トリガーワード
選手育成, 育成プログラム, 選手開発

## 主な機能

### データ管理
- baseball-player-development-agent 関連データのSQLiteデータベース管理
- CRUD操作の実装
- 検索・フィルタリング機能

### チャットボット機能
- Discord連携によるインタラクティブ応答
- 自然言語によるクエリ処理
- コマンドパターンマッチング

## 使用方法

### インストール
```bash
cd agents/baseball-player-development-agent
pip install -r requirements.txt
```

### 実行
```bash
python agent.py
```
