# baseball-nutrition-manager-agent

## 概要
野球栄養管理エージェント。選手の栄養管理・食事計画の提供。

## カテゴリ
野球選手健康・メンタル

## トリガーワード
栄養管理, 食事計画, 栄養士

## 主な機能

### データ管理
- baseball-nutrition-manager-agent 関連データのSQLiteデータベース管理
- CRUD操作の実装
- 検索・フィルタリング機能

### チャットボット機能
- Discord連携によるインタラクティブ応答
- 自然言語によるクエリ処理
- コマンドパターンマッチング

## 使用方法

### インストール
```bash
cd agents/baseball-nutrition-manager-agent
pip install -r requirements.txt
```

### 実行
```bash
python agent.py
```
