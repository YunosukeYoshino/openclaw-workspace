# threat-mitigation-agent

## 概要
脅威緩和エージェント。脅威の緩和策・対策の実装。

## カテゴリ
セキュリティ・脅威ハンティング

## トリガーワード
脅威緩和, 緩和策, 脅威対策

## 主な機能

### データ管理
- threat-mitigation-agent 関連データのSQLiteデータベース管理
- CRUD操作の実装
- 検索・フィルタリング機能

### チャットボット機能
- Discord連携によるインタラクティブ応答
- 自然言語によるクエリ処理
- コマンドパターンマッチング

## 使用方法

### インストール
```bash
cd agents/threat-mitigation-agent
pip install -r requirements.txt
```

### 実行
```bash
python agent.py
```
