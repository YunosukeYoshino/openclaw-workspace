# game-addon-manager-agent

## 概要
ゲームアドオンマネージャーエージェント。アドオン・拡張機能の管理。

## カテゴリ
ゲームUGC・モッズ

## トリガーワード
アドオン, 拡張機能, アドオン管理

## 主な機能

### データ管理
- game-addon-manager-agent 関連データのSQLiteデータベース管理
- CRUD操作の実装
- 検索・フィルタリング機能

### チャットボット機能
- Discord連携によるインタラクティブ応答
- 自然言語によるクエリ処理
- コマンドパターンマッチング

## 使用方法

### インストール
```bash
cd agents/game-addon-manager-agent
pip install -r requirements.txt
```

### 実行
```bash
python agent.py
```
