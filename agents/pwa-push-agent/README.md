# pwa-push-agent

## 概要
PWAプッシュ通知エージェント。PWAプッシュ通知の管理・送信。

## カテゴリ
WebAssembly・PWA

## トリガーワード
プッシュ通知, PWA通知, プッシュ

## 主な機能

### データ管理
- pwa-push-agent 関連データのSQLiteデータベース管理
- CRUD操作の実装
- 検索・フィルタリング機能

### チャットボット機能
- Discord連携によるインタラクティブ応答
- 自然言語によるクエリ処理
- コマンドパターンマッチング

## 使用方法

### インストール
```bash
cd agents/pwa-push-agent
pip install -r requirements.txt
```

### 実行
```bash
python agent.py
```
