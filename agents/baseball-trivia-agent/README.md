# baseball-trivia-agent

Baseball Trivia Agent / 野球クイズ・雑学エージェント

Manage baseball trivia, quizzes, and fun facts

## Description (Japanese)

野球クイズ・雑学・面白い事実を管理するエージェント

## Installation

```bash
cd baseball-trivia-agent
pip install -r requirements.txt
```

## Usage

```bash
python agent.py
```

## Features

- SQLite database for data storage
- Discord bot integration
- RESTful API endpoints (planned)
