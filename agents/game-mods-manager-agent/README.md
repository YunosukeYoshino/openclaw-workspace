# game-mods-manager-agent

## 概要
ゲームMODマネージャーエージェント。ゲームMODの管理・配布。

## カテゴリ
ゲームUGC・モッズ

## トリガーワード
MOD管理, ゲームMOD, MOD配布

## 主な機能

### データ管理
- game-mods-manager-agent 関連データのSQLiteデータベース管理
- CRUD操作の実装
- 検索・フィルタリング機能

### チャットボット機能
- Discord連携によるインタラクティブ応答
- 自然言語によるクエリ処理
- コマンドパターンマッチング

## 使用方法

### インストール
```bash
cd agents/game-mods-manager-agent
pip install -r requirements.txt
```

### 実行
```bash
python agent.py
```
