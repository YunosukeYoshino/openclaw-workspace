# baseball-player-transfer-agent

## 概要
野球選手移籍管理エージェント。選手移籍・トレードの管理。

## カテゴリ
野球選手キャリア管理

## トリガーワード
選手移籍, トレード, 移籍管理

## 主な機能

### データ管理
- baseball-player-transfer-agent 関連データのSQLiteデータベース管理
- CRUD操作の実装
- 検索・フィルタリング機能

### チャットボット機能
- Discord連携によるインタラクティブ応答
- 自然言語によるクエリ処理
- コマンドパターンマッチング

## 使用方法

### インストール
```bash
cd agents/baseball-player-transfer-agent
pip install -r requirements.txt
```

### 実行
```bash
python agent.py
```
