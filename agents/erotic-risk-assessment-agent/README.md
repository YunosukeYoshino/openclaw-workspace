# erotic-risk-assessment-agent

## 概要
えっちリスク評価エージェント。コンテンツのリスク評価・分析。

## カテゴリ
えっちコンテンツセーフティ・プライバシー

## トリガーワード
リスク評価, リスク分析, 安全評価

## 主な機能

### データ管理
- erotic-risk-assessment-agent 関連データのSQLiteデータベース管理
- CRUD操作の実装
- 検索・フィルタリング機能

### チャットボット機能
- Discord連携によるインタラクティブ応答
- 自然言語によるクエリ処理
- コマンドパターンマッチング

## 使用方法

### インストール
```bash
cd agents/erotic-risk-assessment-agent
pip install -r requirements.txt
```

### 実行
```bash
python agent.py
```
