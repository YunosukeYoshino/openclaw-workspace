# baseball-mental-health-agent

## 概要
野球メンタルヘルスエージェント。選手のメンタルヘルス・心理状態の管理。

## カテゴリ
野球選手健康・メンタル

## トリガーワード
メンタルヘルス, 心理状態, メンタルケア

## 主な機能

### データ管理
- baseball-mental-health-agent 関連データのSQLiteデータベース管理
- CRUD操作の実装
- 検索・フィルタリング機能

### チャットボット機能
- Discord連携によるインタラクティブ応答
- 自然言語によるクエリ処理
- コマンドパターンマッチング

## 使用方法

### インストール
```bash
cd agents/baseball-mental-health-agent
pip install -r requirements.txt
```

### 実行
```bash
python agent.py
```
