# erotic-content-review-agent

## 概要
えっちコンテンツレビューエージェント。コンテンツのレビュー・審査。

## カテゴリ
えっちコンテンツセーフティ・プライバシー

## トリガーワード
コンテンツレビュー, 審査, 品質チェック

## 主な機能

### データ管理
- erotic-content-review-agent 関連データのSQLiteデータベース管理
- CRUD操作の実装
- 検索・フィルタリング機能

### チャットボット機能
- Discord連携によるインタラクティブ応答
- 自然言語によるクエリ処理
- コマンドパターンマッチング

## 使用方法

### インストール
```bash
cd agents/erotic-content-review-agent
pip install -r requirements.txt
```

### 実行
```bash
python agent.py
```
