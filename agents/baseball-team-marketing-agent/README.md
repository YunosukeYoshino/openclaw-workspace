# baseball-team-marketing-agent

## 概要
野球チームマーケティングエージェント。チームマーケティングの企画・実行。

## カテゴリ
野球チーム戦略・オペレーション

## トリガーワード
チームマーケティング, マーケティング, プロモーション

## 主な機能

### データ管理
- baseball-team-marketing-agent 関連データのSQLiteデータベース管理
- CRUD操作の実装
- 検索・フィルタリング機能

### チャットボット機能
- Discord連携によるインタラクティブ応答
- 自然言語によるクエリ処理
- コマンドパターンマッチング

## 使用方法

### インストール
```bash
cd agents/baseball-team-marketing-agent
pip install -r requirements.txt
```

### 実行
```bash
python agent.py
```
