# baseball-performance-tracker-agent

## 概要
野球パフォーマンストラッカーエージェント。選手のパフォーマンスデータの追跡・分析。

## カテゴリ
野球デジタル・スマートスタジアム

## トリガーワード
パフォーマンス追跡, 選手データ, パフォーマンス分析

## 主な機能

### データ管理
- baseball-performance-tracker-agent 関連データのSQLiteデータベース管理
- CRUD操作の実装
- 検索・フィルタリング機能

### チャットボット機能
- Discord連携によるインタラクティブ応答
- 自然言語によるクエリ処理
- コマンドパターンマッチング

## 使用方法

### インストール
```bash
cd agents/baseball-performance-tracker-agent
pip install -r requirements.txt
```

### 実行
```bash
python agent.py
```
