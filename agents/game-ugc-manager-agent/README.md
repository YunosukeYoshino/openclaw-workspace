# game-ugc-manager-agent

## 概要
ゲームUGCマネージャーエージェント。ユーザー生成コンテンツの管理・キュレーション。

## カテゴリ
ゲームUGC・モッズ

## トリガーワード
UGC管理, ユーザー生成コンテンツ, UGCキュレーション

## 主な機能

### データ管理
- game-ugc-manager-agent 関連データのSQLiteデータベース管理
- CRUD操作の実装
- 検索・フィルタリング機能

### チャットボット機能
- Discord連携によるインタラクティブ応答
- 自然言語によるクエリ処理
- コマンドパターンマッチング

## 使用方法

### インストール
```bash
cd agents/game-ugc-manager-agent
pip install -r requirements.txt
```

### 実行
```bash
python agent.py
```
