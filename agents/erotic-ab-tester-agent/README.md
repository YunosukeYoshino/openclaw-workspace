# erotic-ab-tester-agent

## 概要
えっちコンテンツA/Bテストエージェント。A/Bテストの実施・分析。

## カテゴリ
えっちコンテンツ品質・ユーザーリサーチ

## トリガーワード
A/Bテスト, テスト実施, ABテスト

## 主な機能

### データ管理
- erotic-ab-tester-agent 関連データのSQLiteデータベース管理
- CRUD操作の実装
- 検索・フィルタリング機能

### チャットボット機能
- Discord連携によるインタラクティブ応答
- 自然言語によるクエリ処理
- コマンドパターンマッチング

## 使用方法

### インストール
```bash
cd agents/erotic-ab-tester-agent
pip install -r requirements.txt
```

### 実行
```bash
python agent.py
```
