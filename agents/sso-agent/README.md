# sso-agent

## 概要
シングルサインオンエージェント。SSOの管理・運用。

## カテゴリ
セキュリティアクセス管理

## トリガーワード
SSO, シングルサインオン, SSO管理

## 主な機能

### データ管理
- sso-agent 関連データのSQLiteデータベース管理
- CRUD操作の実装
- 検索・フィルタリング機能

### チャットボット機能
- Discord連携によるインタラクティブ応答
- 自然言語によるクエリ処理
- コマンドパターンマッチング

## 使用方法

### インストール
```bash
cd agents/sso-agent
pip install -r requirements.txt
```

### 実行
```bash
python agent.py
```
