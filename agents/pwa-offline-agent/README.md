# pwa-offline-agent

## 概要
PWAオフラインエージェント。PWAのオフライン機能・キャッシュ管理。

## カテゴリ
WebAssembly・PWA

## トリガーワード
オフライン, PWAオフライン, キャッシュ

## 主な機能

### データ管理
- pwa-offline-agent 関連データのSQLiteデータベース管理
- CRUD操作の実装
- 検索・フィルタリング機能

### チャットボット機能
- Discord連携によるインタラクティブ応答
- 自然言語によるクエリ処理
- コマンドパターンマッチング

## 使用方法

### インストール
```bash
cd agents/pwa-offline-agent
pip install -r requirements.txt
```

### 実行
```bash
python agent.py
```
