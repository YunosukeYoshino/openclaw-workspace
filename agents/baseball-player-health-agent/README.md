# baseball-player-health-agent

## 概要
野球選手健康管理エージェント。選手の健康状態・メディカルデータの管理。

## カテゴリ
野球選手健康・メンタル

## トリガーワード
選手健康, メディカル, 健康管理

## 主な機能

### データ管理
- baseball-player-health-agent 関連データのSQLiteデータベース管理
- CRUD操作の実装
- 検索・フィルタリング機能

### チャットボット機能
- Discord連携によるインタラクティブ応答
- 自然言語によるクエリ処理
- コマンドパターンマッチング

## 使用方法

### インストール
```bash
cd agents/baseball-player-health-agent
pip install -r requirements.txt
```

### 実行
```bash
python agent.py
```
