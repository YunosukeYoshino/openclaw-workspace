# baseball-player-contract-agent

## 概要
野球選手契約管理エージェント。選手契約の管理・交渉。

## カテゴリ
野球選手キャリア管理

## トリガーワード
選手契約, 契約管理, 契約交渉

## 主な機能

### データ管理
- baseball-player-contract-agent 関連データのSQLiteデータベース管理
- CRUD操作の実装
- 検索・フィルタリング機能

### チャットボット機能
- Discord連携によるインタラクティブ応答
- 自然言語によるクエリ処理
- コマンドパターンマッチング

## 使用方法

### インストール
```bash
cd agents/baseball-player-contract-agent
pip install -r requirements.txt
```

### 実行
```bash
python agent.py
```
