# baseball-podcast-agent

Baseball Podcast Agent / 野球ポッドキャスト・音声コンテンツ収集エージェント

Collect and manage baseball podcasts and audio content

## Description (Japanese)

野球ポッドキャスト・音声コンテンツを収集・管理するエージェント

## Installation

```bash
cd baseball-podcast-agent
pip install -r requirements.txt
```

## Usage

```bash
python agent.py
```

## Features

- SQLite database for data storage
- Discord bot integration
- RESTful API endpoints (planned)
