# threat-simulation-agent

## 概要
脅威シミュレーションエージェント。攻撃シミュレーション・テスト。

## カテゴリ
セキュリティ・脅威ハンティング

## トリガーワード
脅威シミュレーション, 攻撃シミュレーション, レッドチーム

## 主な機能

### データ管理
- threat-simulation-agent 関連データのSQLiteデータベース管理
- CRUD操作の実装
- 検索・フィルタリング機能

### チャットボット機能
- Discord連携によるインタラクティブ応答
- 自然言語によるクエリ処理
- コマンドパターンマッチング

## 使用方法

### インストール
```bash
cd agents/threat-simulation-agent
pip install -r requirements.txt
```

### 実行
```bash
python agent.py
```
