# baseball-highlights-agent

Baseball Highlights Video Agent / 野球ハイライト映像管理エージェント

Manage baseball game highlight videos and clips

## Description (Japanese)

野球試合のハイライト映像・クリップを管理するエージェント

## Installation

```bash
cd baseball-highlights-agent
pip install -r requirements.txt
```

## Usage

```bash
python agent.py
```

## Features

- SQLite database for data storage
- Discord bot integration
- RESTful API endpoints (planned)
