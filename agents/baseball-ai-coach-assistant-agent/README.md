# baseball-ai-coach-assistant-agent

## 概要
野球AIコーチアシスタントエージェント。AIによるコーチング支援・アドバイスの提供。

## カテゴリ
野球デジタル・スマートスタジアム

## トリガーワード
AIコーチ, コーチアシスタント, AIアドバイス

## 主な機能

### データ管理
- baseball-ai-coach-assistant-agent 関連データのSQLiteデータベース管理
- CRUD操作の実装
- 検索・フィルタリング機能

### チャットボット機能
- Discord連携によるインタラクティブ応答
- 自然言語によるクエリ処理
- コマンドパターンマッチング

## 使用方法

### インストール
```bash
cd agents/baseball-ai-coach-assistant-agent
pip install -r requirements.txt
```

### 実行
```bash
python agent.py
```
