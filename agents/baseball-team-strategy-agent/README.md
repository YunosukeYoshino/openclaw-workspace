# baseball-team-strategy-agent

## 概要
野球チーム戦略エージェント。チーム戦略の立案・分析。

## カテゴリ
野球チーム戦略・オペレーション

## トリガーワード
チーム戦略, 戦略立案, チーム戦略分析

## 主な機能

### データ管理
- baseball-team-strategy-agent 関連データのSQLiteデータベース管理
- CRUD操作の実装
- 検索・フィルタリング機能

### チャットボット機能
- Discord連携によるインタラクティブ応答
- 自然言語によるクエリ処理
- コマンドパターンマッチング

## 使用方法

### インストール
```bash
cd agents/baseball-team-strategy-agent
pip install -r requirements.txt
```

### 実行
```bash
python agent.py
```
