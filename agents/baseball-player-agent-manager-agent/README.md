# baseball-player-agent-manager-agent

## 概要
野球選士エージェント。選手エージェントの業務・管理。

## カテゴリ
野球選手キャリア管理

## トリガーワード
選士エージェント, エージェント業務, 選手マネージメント

## 主な機能

### データ管理
- baseball-player-agent-manager-agent 関連データのSQLiteデータベース管理
- CRUD操作の実装
- 検索・フィルタリング機能

### チャットボット機能
- Discord連携によるインタラクティブ応答
- 自然言語によるクエリ処理
- コマンドパターンマッチング

## 使用方法

### インストール
```bash
cd agents/baseball-player-agent-manager-agent
pip install -r requirements.txt
```

### 実行
```bash
python agent.py
```
