# abac-agent

## 概要
属性ベースアクセス制御エージェント。ABACの管理・運用。

## カテゴリ
セキュリティアクセス管理

## トリガーワード
ABAC, 属性ベース, 属性制御

## 主な機能

### データ管理
- abac-agent 関連データのSQLiteデータベース管理
- CRUD操作の実装
- 検索・フィルタリング機能

### チャットボット機能
- Discord連携によるインタラクティブ応答
- 自然言語によるクエリ処理
- コマンドパターンマッチング

## 使用方法

### インストール
```bash
cd agents/abac-agent
pip install -r requirements.txt
```

### 実行
```bash
python agent.py
```
