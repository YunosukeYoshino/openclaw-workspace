# game-data-science-agent

## 概要
ゲームデータサイエンスエージェント。データ分析・インサイト生成。

## カテゴリ
ゲームAI・MLモデルトレーニング

## トリガーワード
データサイエンス, データ分析, インサイト

## 主な機能

### データ管理
- game-data-science-agent 関連データのSQLiteデータベース管理
- CRUD操作の実装
- 検索・フィルタリング機能

### チャットボット機能
- Discord連携によるインタラクティブ応答
- 自然言語によるクエリ処理
- コマンドパターンマッチング

## 使用方法

### インストール
```bash
cd agents/game-data-science-agent
pip install -r requirements.txt
```

### 実行
```bash
python agent.py
```
