# baseball-player-career-agent

## 概要
野球選手キャリア管理エージェント。選手のキャリア全般の管理・追跡。

## カテゴリ
野球選手キャリア管理

## トリガーワード
選手キャリア, キャリア管理, 選手経歴

## 主な機能

### データ管理
- baseball-player-career-agent 関連データのSQLiteデータベース管理
- CRUD操作の実装
- 検索・フィルタリング機能

### チャットボット機能
- Discord連携によるインタラクティブ応答
- 自然言語によるクエリ処理
- コマンドパターンマッチング

## 使用方法

### インストール
```bash
cd agents/baseball-player-career-agent
pip install -r requirements.txt
```

### 実行
```bash
python agent.py
```
