# data-lake-agent

## 概要
データレイクエージェント。データレイクの管理・運用。

## カテゴリ
データウェアハウス・データレイク

## トリガーワード
データレイク, レイク, データストレージ

## 主な機能

### データ管理
- data-lake-agent 関連データのSQLiteデータベース管理
- CRUD操作の実装
- 検索・フィルタリング機能

### チャットボット機能
- Discord連携によるインタラクティブ応答
- 自然言語によるクエリ処理
- コマンドパターンマッチング

## 使用方法

### インストール
```bash
cd agents/data-lake-agent
pip install -r requirements.txt
```

### 実行
```bash
python agent.py
```
