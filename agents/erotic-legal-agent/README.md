# erotic-legal-agent

## 概要
えっちコンテンツ法務エージェント。法務対応・契約・紛争解決。

## カテゴリ
えっちコンテンツ法務・コンプライアンス

## トリガーワード
法務, 契約, 紛争解決

## 主な機能

### データ管理
- erotic-legal-agent 関連データのSQLiteデータベース管理
- CRUD操作の実装
- 検索・フィルタリング機能

### チャットボット機能
- Discord連携によるインタラクティブ応答
- 自然言語によるクエリ処理
- コマンドパターンマッチング

## 使用方法

### インストール
```bash
cd agents/erotic-legal-agent
pip install -r requirements.txt
```

### 実行
```bash
python agent.py
```
