# observability-monitor-agent

## 概要
オブザーバビリティモニターエージェント。システムの可視化・監視。

## カテゴリ
オブザーバビリティ・モニタリング

## トリガーワード
オブザーバビリティ, 可視化, 監視

## 主な機能

### データ管理
- observability-monitor-agent 関連データのSQLiteデータベース管理
- CRUD操作の実装
- 検索・フィルタリング機能

### チャットボット機能
- Discord連携によるインタラクティブ応答
- 自然言語によるクエリ処理
- コマンドパターンマッチング

## 使用方法

### インストール
```bash
cd agents/observability-monitor-agent
pip install -r requirements.txt
```

### 実行
```bash
python agent.py
```
