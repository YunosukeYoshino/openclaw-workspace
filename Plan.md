### 優先順位 5: ゲームソーシャルメタゲーミングエージェント / Game Social Metagaming Agents
**必要エージェント数**: 5個
**概要 (Overview)**: ゲームのメタゲーム（戦略分析、環境推移、Tierリスト）をコミュニティと共有し、議論・投票するためのソーシャルエージェント群。

**エージェント構成**:
1. **game-meta-reporter-agent** - ゲームメタレポーターエージェント / Game Meta Reporter Agent
   - 最新の環境推移、人気キャラ・デッキの追跡
   - プロプレイヤー、トーナメント結果の分析
   - 定期的なメタレポートの生成・配信

2. **game-tier-list-manager-agent** - ゲームTierリスト管理エージェント / Game Tier List Manager Agent
   - コミュニティ投票によるTierリストの作成・更新
   - バージョン別、ロール別、使用率別のTierリスト
   - 履歴管理、トレンド表示、差分比較

3. **game-counter-pick-agent** - ゲームカウンターピックエージェント / Game Counter Pick Agent
   - 特定キャラ・デッキへの最適カウンターピック提案
   - シナジー、アンチシナジーの分析
   - ビルドガイド、戦略解説の自動生成

4. **game-meta-discussion-agent** - ゲームメタ議論エージェント / Game Meta Discussion Agent
   - メタ環境に関する議論スレッドの自動生成
   - 注目トピック、投票、アンケートの作成
   - コミュニティ意見の集約、サマリー表示

5. **game-build-optimizer-agent** - ゲームビルド最適化エージェント / Game Build Optimizer Agent
   - アイテム、スキル、ルーンの最適ビルド提案
   - 対戦相手、状況別のビルド変化案
   - パッチ対応、成功率統計、ビルド共有
