# baseball-historical-match-agent

## Description
Baseball Historical Match Agent - 野球歴史的名試合エージェント

This agent provides comprehensive functionality for tracking and managing historical baseball matches.
Includes SQLite database support.

## Features

- **Match Tracking**: Store and manage historical baseball matches
- **Multiple Match Types**: World Series, All-Star games, playoffs, regular season
- **Importance Levels**: Categorize matches as legendary, historic, or memorable
- **Key Players & Moments**: Track important players and defining moments
- **Records Broken**: Monitor records set during matches
- **Statistics**: View database statistics and insights

## Installation

1. Install required dependencies:
```bash
pip install -r requirements.txt
```

2. Set up environment variables (optional):
```bash
export DISCORD_BOT_TOKEN="your_discord_bot_token_here"
```

## Usage

### Command Line

```bash
# Add a historical match
python3 agent.py add date="1924-10-10" home_team="Washington Senators" away_team="New York Giants"

# Get a match
python3 agent.py get id=1

# List matches
python3 agent.py list

# Get stats
python3 agent.py get_stats
```

### Discord Bot

```bash
# Start Discord bot
python3 discord.py --token YOUR_TOKEN

# Available commands:
!help - Show help
!add - Add a historical match
!get <id> - Get a match
!list [type] - List matches
!search <query> - Search matches
!stats - Show statistics
```

## Match Types

- `world_series`: World Series games
- `all_star`: All-Star games
- `playoff`: Playoff games
- `regular_season`: Regular season games

## Importance Levels

- `legendary`: Most iconic matches
- `historic`: Historically significant matches
- `memorable`: Memorable matches
