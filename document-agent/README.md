# document-agent

## Description
Document Agent - ドキュメント管理エージェント

This agent provides comprehensive functionality for document storage and management with natural language support.
Includes SQLite database support and Discord integration.

## Features

- **Document Management**: Add, update, list, and delete documents
- **Multi-language Support**: Automatic language detection (Japanese/English)
- **Category System**: Organize documents by category
- **Tag System**: Tag documents for easy retrieval
- **Search**: Full-text search across documents
- **Version History**: Track document revisions
- **Export/Import**: Export documents to JSON/Markdown formats
- **Discord Integration**: Full Discord bot interface with commands
- **Statistics**: View database statistics and insights

## Installation

1. Install required dependencies:
```bash
pip install -r requirements.txt
```

2. Set up environment variables (optional):
```bash
export DISCORD_BOT_TOKEN="your_discord_bot_token_here"
```

## Usage

### Command Line

```bash
# Add a document
python3 agent.py add title="My Document" content="Document content" category="notes"

# Get a document
python3 agent.py get id=1

# List documents
python3 agent.py list

# Search documents
python3 agent.py search "baseball"

# Update status
python3 agent.py update_status id=1 status=archived

# Get stats
python3 agent.py get_stats

# Export documents
python3 agent.py export format=json
```

### Discord Bot

```bash
# Start Discord bot
python3 discord.py --token YOUR_TOKEN

# Available commands:
!help - Show help
!add <title> <content> - Add a document
!get <id> - Get a document
!list [category] - List documents
!search <query> - Search documents
!update <id> <status> - Update document status
!delete <id> - Delete a document
!stats - Show statistics
!categories - List categories
!tags - List tags
!export <format> - Export documents (json, markdown)
```

## Languages

- `ja`: Japanese documents
- `en`: English documents

## Statuses

- `active`: Currently active
- `archived`: Archived but not deleted
- `deleted`: Soft deleted

## Export Formats

- `json`: JSON format with all metadata
- `markdown`: Markdown format with frontmatter
