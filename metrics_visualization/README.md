# Metrics Visualization / メトリクス可視化

システムメトリクスのリアルタイム可視化

## 機能

- リアルタイムメトリクス収集
- CPU使用率の追跡
- メモリ使用量の追跡
- ディスク使用量の追跡
- ネットワークトラフィックの追跡
- メトリクス履歴の保存

## 使い方

```bash
python3 implementation.py
```

## 依存パッケージ

```
psutil
```
